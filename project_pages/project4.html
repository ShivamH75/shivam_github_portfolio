Tableau: Manufacturing Defect Analysis

Overview

This project aims to create a comprehensive Power BI dashboard for analyzing electric vehicle (EV) data. The dataset includes detailed information on electric vehicles, such as their location, make, model, type, and various other attributes. The dashboard will provide insights into the distribution, characteristics, and trends of electric vehicles across different counties and cities.

Dataset

The data for this project is sourced from Kaggle (Ref. dataset).

The dataset contains the following columns:

Sr. No. Entity Description
1. defect_id Unique identifier for each defect.
2. product_id Identifier for the product associated with the defect.
3. defect_type Type of defect (e.g., Structural, Functional, Cosmetic).
4. defect_date Date when the defect was recorded.
5. defect_location Location of the defect (e.g., Component, Internal, Surface).
6. severity Severity level of the defect (e.g., Minor, Moderate, Critical).
7. inspection_method Method used to inspect the defect (e.g., Visual Inspection, Automated Testing, Manual Testing).
8. Model The specific model of the vehicle
9. repair_cost Cost to repair the defect.

Initial Analysis

Based on the dataset, we can perform the following analyses: Defect Distribution by Type: Analyze the distribution of different types of defects. Defect Severity Analysis: Analyze the severity levels of the defects. Defect Location Analysis: Analyze where defects are commonly found. Inspection Method Effectiveness: Compare the effectiveness of different inspection methods. Repair Cost Analysis: Analyze the repair costs associated with different defects. Trend Analysis: Analyze the trends over time for defect occurrences.

Expected Outcomes

  • A user-friendly Power BI dashboard providing valuable insights into the distribution and characteristics of electric vehicles.

  • Enhanced decision-making capabilities for policymakers and stakeholders based on comprehensive EV data analysis.

  • Identification of trends and patterns in electric vehicle adoption, helping to inform future strategies for promoting clean transportation.
