SQL Data Analysis: Air Quality Index

Project Summary

SQL project executed within Jupyter Notebooks using Python. By leveraging the versatility of Jupyter Notebooks and the power of SQL, this project aims to provide an interactive platform for SQL queries and database management.

Introduction

The Air Quality Index (AQI) is a crucial metric used to communicate the quality of the air and its potential health impacts. This project aims to analyze air quality data across various cities, states, and pollutants to derive meaningful insights and trends. The dataset includes information on pollutant levels such as PM10 and PM2.5, recorded at different monitoring stations across multiple locations.

Dataset

The data for this project is sourced from data.gov.in, an open data platform that provides access to various datasets from the Indian government. This project aims to utilize real-time data to derive meaningful insights and trends (Ref. dataset). The csv data is imported into MySQL and obtained a database file to begin the analysis.

Sr. No. Entity Description
1. country The country where the data is recorded.
2. state The state within the country.
3. city The specific city where the monitoring station is located.
4. station The name of the monitoring station.
5. last_update The timestamp of the last recorded data.
6. latitude The latitude of the monitoring station.
7. longitude The longitude of the monitoring station.
8. pollutant_id The type of pollutant (e.g., PM10, PM2.5).
9. pollutant_min The minimum recorded level of the pollutant.
10. pollutant_max The maximum recorded level of the pollutant.
11. pollutant_avg The average recorded level of the pollutant.

SQL Analysis Questions

    1. What is the average pollutant level (pollutant_avg) for each pollutant type across all cities?
    2. Which city has the highest average pollutant level for PM10?
    3. How does the average pollutant level of PM2.5 compare between different states?
    4. What is the trend of pollutant levels over time for a specific city?
    5. Which stations have recorded the highest maximum pollutant levels (pollutant_max) for each pollutant?
    6. What is the average pollutant level for each city for a specific pollutant (e.g., PM2.5) over the past month?
    7. Are there any significant differences in pollutant levels between urban and rural stations?
    8. Which pollutants have shown the most variation in their levels (difference between pollutant_max and pollutant_min) across different stations?
    9. How do pollutant levels vary by geographical coordinates (latitude and longitude)?
    10. Which city has the highest average pollutant level for PM10 and PM2.5 combined?

Insights and Conclusions

The analysis will provide valuable insights into the air quality situation across different regions. Key findings will highlight areas with the highest pollution levels, identify trends over time, and uncover significant differences between pollutant types and their impacts. These insights can inform public health recommendations, policy decisions, and environmental initiatives aimed at improving air quality.
